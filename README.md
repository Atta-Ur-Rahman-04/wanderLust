# WanderLust is a full-stack Airbnb-inspired web application where users can browse, list, and book unique places to stay. It's built with the MERN-style stack (MongoDB, Express, Node.js) using EJS for templating, and includes real-world features like image uploads, interactive maps, and secure user authentication.

# I built this project to get hands-on experience with full-stack development — everything from designing the database schema to handling authentication, file uploads, and deployment.

## Features

- **Browse & Search Listings** — View all available stays and search listings by country
- **User Authentication** — Secure sign up, login, and logout using Passport.js
- **Create & Manage Listings** — Logged-in users can add, edit, and delete their own listings
- **Image Uploads** — Listing photos are uploaded and stored using Cloudinary
- **Interactive Maps** — Each listing displays its location on a map powered by Mapbox
- **Reviews & Ratings** — Users can leave reviews and star ratings on listings
- **Flash Messages** — Friendly success/error feedback after actions like login, booking, or errors
- **Responsive Design** — Built with Bootstrap for a clean experience across devices

## Tech Stack

**Frontend:** EJS, Bootstrap, CSS
**Backend:** Node.js, Express.js
**Database:** MongoDB (Mongoose)
**Authentication:** Passport.js
**Image Storage:** Cloudinary
**Maps:** Mapbox
